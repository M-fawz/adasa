# Adasa (عدسة) — Photography Blog Platform

Pixel-perfect clone based on reference UI + scalable architecture.
Arabic/English (RTL/LTR) supported from day 1.

## Live Reference
- **Reference UI**: [https://adasa-psi.vercel.app/blog](https://adasa-psi.vercel.app/blog)

## Tech Stack
- **Framework**: React + Vite
- **Routing**: React Router v6
- **Styling**: TailwindCSS
- **State Management**: Zustand
- **i18n**: react-i18next

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
